## Key learnings

### FreqLeader (433 lines, 5 phases + failed)
Meet-and-Lead with nar hooks. State machine:
```
MEET → LEAD → HOLD → LEAD → SUSTAIN
             → FAILED (max_hold timeout, micro-step back)
             → SUSTAIN dropout: micro-step back to LEAD
```
- **Passthrough mode**: When control panel agent drives beat_frequency directly (conductor passthrough), the leader publishes freq_lead_data recommendations only — never overwrites beat_frequency. Keeps the two agents from fighting.
- **Fast-follower mode**: 3 holds-free consecutive steps → halve step_interval (floor 15s). Compresses descent without breaking the hold safety net.
- **Nar integration**: `_narrate()` writes `agent_message` to live_control.json. In passthrough mode it skips (line 182-194). Uses `_narrated_phases` set to prevent repeat utterances, `force=True` to override.
- **Deactivation**: Restores beat_frequency to `eeg_iaf_hz` (falling back to current beat). Doesn't snap to a hard-coded default.
- **Alpha-overlap correction**: When `entrainment_confidence == "alpha_overlap"`, scales ASSR × 0.7 before phase transition.
- **FAILED/FAILED-LIKE**: Only `max_hold` timeout exists — pushes current_freq up by step_size, resets timer, no "give up" terminal state. Holds loop indefinitely.
- **Sustain strategy**: Enabled via `freq_lead_mode == "sustain"`; skips MEET entirely.

### PAC Estimator (156 lines, Tort 2010)
- **MI metric**: KL-divergence of amplitude-binned-by-phase vs uniform, normalized by log(N_bins=18).
- **Three pairs**: ISA-alpha (40% weight, 30s buffer, 0.03–0.08 Hz phase → IAF±2 amp), theta-gamma (35%), alpha-gamma (25%).
- **ISA**: Needs full 30 s buffer for infra-slow phase. Falls back to short 4 s window with raised floor (0.05–0.12 Hz) during first 30 seconds — returns 0.0 on that fall if buffer < window_len.
- Rate-limited at update interval 2 s; returning empty dict.
- `compute_all()` returns 4 keys: `pac_isa_alpha`, `pac_theta_gamma`, `pac_alpha_gamma`, `pac_cascade_integrity`.
- Called from `eeg_engine.py` at ~0.5 Hz.

- **Shader assembly**: `_assemble_shader()` reads `common.glsl`, then each `style_*.glsl` from `shaders/styles/`, then appends 30-entry if/else dispatcher. One monolithic program — changing any style requires `rebuild_program()`.

- **IAF rotation lock**: Recomputes target every 30 s, 2 s lerp. Converts Hz to speed_mult via `/ 2.5` baseline (10 Hz IAF / 4 arms = 2.5 Hz → 1.0 mod). Basketball in Bible Ch.10 note.
- **Motion aftereffect**: Three-phase (running → ramp-down → pause → flip → ramp-up). `ae_direction` and `ae_ramp_factor` together form a 20 state per-tick return value that doesn't lock out the step function.
- **Loom oscillator**: Sync sources — manual rate, breath (`respiratory_rate_hz`), beat_div4 (`beat_frequency / 4`). Sine wave onto radial scale: `1.0 + depth * 0.25 * sin(loom_accum * tau)`.
